# Digitalbank Landing Page

A responsive landing page for a digital banking product, built with **SvelteKit** and **Sanity CMS**. Content is managed in Sanity and rendered dynamically on the frontend.

## Tech Stack

- **Frontend:** SvelteKit (Svelte 5)
- **CMS:** Sanity
- **Challenge:** [Frontend Mentor - Digitalbank](https://www.frontendmentor.io/challenges/digital-bank-landing-page-WaUhkoDN)

## Quick Start

```bash
# Install dependencies
npm install

# Run frontend (port 5173)
npm run dev
```

```bash
# Run Sanity Studio (port 3333)
cd studio-digitalbank && npm run dev
```

Add `PUBLIC_SANITY_PROJECT_ID` and `PUBLIC_SANITY_DATASET` to `.env` in the project root.
